# nfl_draft

Simple 2026 NFL Draft simulator and team pick lookup.

## Run a full 7-round simulation (224 picks)

```bash
python /home/runner/work/nfl_draft/nfl_draft/nfl_draft.py
```

## View picks for a single team (example: Dallas Cowboys)

```bash
python /home/runner/work/nfl_draft/nfl_draft/nfl_draft.py --team "Dallas Cowboys"
```

## Run tests

```bash
cd /home/runner/work/nfl_draft/nfl_draft
python -m unittest discover -s tests -v
```
